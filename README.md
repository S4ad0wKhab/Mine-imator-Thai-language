# Mine-imator ภาษาไทย (2.0.1)
ไฟล์ภาษาไทยสำหรับ Mine-imator โดย S4ad0wKhab
# ก่อนจะมีโปรเจกต์นี้
Mine-imator ปัจจุบันมีการแปลภาษาไทยสำหรับเวอร์ชัน 1.0.0 DEMO 5 ถูกแปลไปแล้ว 97% โดย NewlyKunG ถูกเผยแพร่บน Mine-imator forums [ดูฟอรั่ม](https://www.mineimatorforums.com/index.php?/topic/43493-mine-imator-100-demo-5-in-thai/) 
หลังจากนั้นมีการแปลภาษาไทยสำหรับเวอร์ชัน 1.0.6 ถูกแปลไปแล้ว 37% โดย Kongpha แต่ก็หยุดพัฒนาไปจน Mine-imator ได้ประกาศยุติการพัฒนาและประกาศเปิดตัว Mine-imator 2.0 หลังจากนั้นต่อมาก็ไม่มีการแปลภาษาไทยอีกเลยจนมาถึงโปรเจกต์นี้!
